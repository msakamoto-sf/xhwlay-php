method gc [line 582]

void gc( )

Simply comparing filemtime() for each Bookmark data file.

This method generate INFO level message which contains data directory, total count of datafiles, count of destroyed files.

How detecting expiration data is shown in following actual source code.

  1. function gc()
  2.     {
  3.         $now = time();
  4.         $files = File_Util::listDir($this->_dataDir, FILE_LIST_FILES);
  5.         $target = array();
  6.         foreach ($files as $file) {
  7.             $mtime = $file->date;
  8.             if ($now - $mtime > $this->_gc_maxlifetime) {
  9.                 // Current time - last modified time is over against
  10.                 // TTL, then delete bookmark data file.
  11.                 $target[] = $file->name;
  12.                 @unlink($this->fileName($file->name));
  13.             }
  14.         }
  15.         Xhwlay_ErrorStack::push(
  16.             XHWLAY_BOOKMARK_CONTAINER_EC_GC_INVOKED,
  17.             "Bookmark Data Garbage Collection was Invoked.",
  18.             XHWLAY_ERRORSTACK_EL_INFO,
  19.             array(
  20.                 "dataDir" => $this->_dataDir,
  21.                 "total" => count($files),
  22.                 "target" => $target,
  23.             )
  24.         );
  25.     }
