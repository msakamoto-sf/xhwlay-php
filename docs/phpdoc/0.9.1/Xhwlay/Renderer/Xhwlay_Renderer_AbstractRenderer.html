[line 58]

AbstractRenderer of Xhwlay Renderer Classes.

This class implements "Template Variables Manupilation" methods.  But core "render()" methods is not implemented yet. Only definition.
So, developers must create class extend this, and implements  Xhwlay_Renderer_AbstractRenderer::render() method.
