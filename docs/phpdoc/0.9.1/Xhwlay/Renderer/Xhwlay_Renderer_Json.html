Class Details

[line 59]
JSON output renderer

It requires Jsphon package(pear.hawklab.jp), PHP mb_string() module.

By default, don't convert to UTF-8. If you output JSON as UTF-8, your code like below:

  1.  $renderer =& new Xhwlay_Renderer_Json();
  2.  $renderer->autoUTF8 = true;
  3.  $renderer->escapeNonASCII = true;
  4.  $renderer->set('key1', "(multibye character)");
  5.  
  6.  echo $renderer->render();
