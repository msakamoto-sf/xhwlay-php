static method diceRoller [line 101]

	

    

    

		static boolean diceRoller(
integer
$probability, integer
$divisor)
    

    


	
		Get true or false by probability.

Return true in $probability/$divisor percent probability.  Example:  
  1.  Xhwlay_Util::diceRoller(1, 100)
    2. 

  This return true one time in 100 calls.  99 times in 100 calls return false.
If $probability = 0, always return false.  If $probability >= $divisor, always return true.
