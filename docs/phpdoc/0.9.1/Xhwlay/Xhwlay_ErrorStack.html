static method pushCallback [line 165]

	

    

    

		static void pushCallback(
callback
$callback)
    

    


	
		Pushes a callback. This method is a wrapper for  PEAR_ErrorStack::staticPushCallback() method.

$callback method/function is like below:  
  1.  function XXXX($error) // 1 arg is error info(assoc-array)
    2. 

  2.  {
    3. 

  3.   ... (your own codes) ...
    4. 

  4.   return PEAR_ERRORSTACK_IGNORE; // Don't stack, ignored.
    5. 

  5.   return PEAR_ERRORSTACK_PUSH; // Stack.
    6. 

  6.   return PEAR_ERRORSTACK_DIE; // die().
    7. 

  7.  }
    8.
